IClick Class

Defined in: src/IClick.js:33

  • config
)
static

Defined in src/IClick.js:55

Adds device support

Parameters:

  • name String

    name of device

  • config Object

    device configurations

    • start_event String

      name of start event

    • event_map Objcet

      a map of event names and their phase name. eg: { mousedown : 'STARTEVENT', mousemove : 'MOVEEVENT', mouseup : 'END_EVENT' }

    • handlers Objcet

      map of event handlers. basic structure:

           type : {
      INITIAL : {
           TYPE : {
              CAPTURING : fn(ev,_ev){}, //optional
              BUBBLING : fn(ev,_ev){} //optional
           }
      },
      ACTIVE : {
           TYPE : {
               CAPTURING : fn(ev,_ev){}, //optional
               BUBBLING : fn(ev,_ev){} //optional
           }
      }
    }
